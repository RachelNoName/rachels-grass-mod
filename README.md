# Rachels Grass Mod
Ads a few grass related blocks and items. Inspired by the grass cutting mechanic from [GregTech 6](https://gregtech.overminddl1.com/).

# Features
(Note: The mod is still HEAVILY in development so almost none of these are added yet. This list is currently more like a list of planned features. I am working on it!)   

- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Adds new unique grass related blocks and items.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Provides a way to cut grass using swords or knives.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Grass can be crafted into a unique block.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) The grass block can dry if placed in the sun or become wet if placed in rain.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Grass can also become wet by using a water bottle or dried by smelting in a furnace.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Normal grass can be composted and dry grass can be used as furnace fuel and to make campfires instead of coal.  
- ![Grass](src/main/resources/assets/rachels_grass/textures/item/grass_bundle.png) Grass can be painted to prevent it from drying or becoming wet (usefull for building purposes).
